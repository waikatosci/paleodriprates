Package motabar.

Provides classes for regression/interpolation/smoothing using the MoTaBaR method (Moving Taylor Bayesian Regression).

Related publications:

  • J. Heitzig: Moving Taylor Bayesian Regression. Working Paper. Potsdam Institute for Climate Impact Research (PIK), 2011.

Author: Jobst Heitzig <heitzig@pik-potsdam.de>

Requires: Python 2.6.5+, Numpy, Scipy, progressbar

Version: 0.9

See: The MoTaBaR webpage
