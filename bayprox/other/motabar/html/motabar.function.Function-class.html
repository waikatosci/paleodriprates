set_priors(self, mf=0, mr=0, sf=inf, sr=None, s0=None, omega=None, l=None, corr1=0, corr2=0, verbosity=None)

source code 

Set prior distributions for phi and psi

If covariances are not given, a standard correlation structure is used: The correlation between the alpha-th and beta-th derivatives is corr1 * corr2**abs(|alpha|-|beta|-1) if |alpha|-|beta| is odd and corr2**abs(|alpha|-|beta|) if it is even.

s0 and omega or l can be used to generate standard deviations of the form s0 * omega**alpha or s0 * (2*pi/l)**alpha

Parameters:
  • mf (1d array or float) - Prior mean of phi. Default: 0.
  • mr (1d array or float) - Common prior mean of all psi[i]. Default: 0.
  • sf (1d or 2d array, or inf) - If 1d, prior standard deviations of phi. If 2d, prior covariance matrix of phi. If inf, a noninformative prior for phi is used. Default: inf
  • sr (1d or 2d array, float, or None) - If float, common prior standard devs. of all psi[i,k]. If 1d, common prior standard devs. of all psi[i]. If 2d, common prior covariance matrix of all psi[i]. If None, will be estimated from the data (see article). Default: None
  • s0 (float > 0) - Prior standard deviation of f(x).
  • omega (1d array or float > 0) - Optional "typical angular frequencies" in each argument. Default: 2*pi / l
  • l (1d array or float > 0) - Optional "typical length scale" in each argument.
  • corr1, corr2 - Correlation structure. For f(x)=exp(ax), corr1=corr2>0 is suitable. For f(x)=sin(ax), corr1=0 and corr2<0 is suitable. Default: corr1=corr2=0 USE WITH CARE!
  • verbosity (int > 0) - Optional verbosity level. Default: as set by class property "verbosity".
  • corr1,corr2 (float in [-1,1])

source code 

Set default error distributions for all subsequently added data.

Standard deviations, correlations, or covariances not specified are taken as 0.

Parameters:
  • sx (1d or 2d array or float #or function(x_i)) - If 1d, standard deviations of each errx[i]. If 2d, covariance matrix of each errx[i]. Default: 0
  • sy (float #or function(x_i)) - Standard deviation of each erry[i]. Default: 0
  • cx (2d array or float #or function(x_i,x_j)) - Correlation matrix between each pair errx[i],errx[j] (i!=j). Default: 0
  • cy (float #or function(x_i,x_j)) - Correlation between each pair erry[i],erry[j] (i!=j). Default: 0
  • verbosity (int > 0) - Optional verbosity level. Default: as set by class property "verbosity".

__call__(self, x, return_w=False, verbosity=None, obs=inf, method="LUpinv")
(Call operator)

source code 

Return posterior mean and precision matrix of f and its derivatives of order <p for one or more arguments of interest.

NOTE: does not consider argument error yet!

Parameters:
  • x (float or 1d or 2d array) - Argument(s) of interest. If d=1 and float, x is one argument of interest. If d>1 and 1d, x[:] is one argument of interest. If d=1 and 1d, each x[i] is one argument of interest. If d>1 and 2d, each x[i,:] is one argument of interest.
  • return_w (bool) - If True, return MoTaBaR coefficient vectors as well. Default: False
  • obs (int>0 or float <1) - If int>0, this many observations will be used in the estimation for each argument of interest. If float<1, as many observations will be used as are necessary to ensure that the remaining observations have a total weight <obs. Default: inf (all observations will be used) USE WITH CARE!
  • method ("LU", "LUpinv", "pinvLU", "pinv", or "lstsq") - Which numerical method to use for solving the involved linear systems of equations. See code for details. Default: "LUpinv"
  • verbosity (int > 0) - Optional verbosity level. Default: as set by class property "verbosity".
Returns: array, array [, array]
Posterior mean and precision matrix of phi. If x contains more than one argument of interest, the first index of each return array is its index. If return_w=True, (each row of) the third array contains a the vector of MoTaBaR coefficients used to derive the posterior mean from the data values.
